연산자 입력

2 입력 시 결과 값 리스트 보여주고 삭제 할 값 입력 시 한개만 삭제
  입력 값을 삭제 한 후 리스트를 다시 보여주고
  switch문 (1.종료) (2.삭제) (3.계산) check 값을 선택하도록 함.
